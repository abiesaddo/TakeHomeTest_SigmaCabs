# Take Home Test – Sigma Cabs Surge Pricing Analysis

## 📌 Deskripsi Proyek
Analisis faktor-faktor yang mempengaruhi Surge Pricing pada layanan taksi Sigma Cabs, menggunakan data trip dan perilaku pelanggan.

## 📁 Struktur File
- `notebook.ipynb` – Google Colab (Python) untuk data cleaning, feature engineering, dan EDA.
- `sigma_cabs_clean.csv` – Dataset final yang sudah dibersihkan.
- `dashboard.pbix` – File Power BI dashboard (3 halaman).
- Slide presentasi Canva, terlampir pada bagian Link Akses

## 🔧 Tools
- Google Colab (Python, Pandas, Matplotlib, Seaborn)
- Power BI Desktop

## 📊 Dashboard
3 halaman dashboard interaktif:
1. Executive Pricing & Fleet Overview
2. Surge Drivers & Operational Analysis
3. Customer Experience & Service Quality

## 📎 Link Akses
- PPT Canva: https://canva.link/uetxde0kjlmlcc0

## Catatan Akses File:
- sigma_cabs.csv adalah dataset mentah yang digunakan dalam proses EDA di Google Colab (menggunakan mounting Google Drive).
- sigma_cabs_clean.csv adalah dataset hasil preprocessing yang langsung terhubung dengan file Power BI (dashboard.pbix).
- Untuk menjalankan file .pbix, unduh sigma_cabs_clean.csv dan tempatkan di folder yang sama dengan file .pbix, atau atur ulang koneksi data di Power BI.
